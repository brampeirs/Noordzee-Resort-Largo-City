There’s more than one way to mockup a website.

Bram Peirs
Writes on October 28, 2013

You’ve probably seen PSD to HTML advertisements around the web. If you type “PSD to HTML” into your favorite search engine, you’ll find lots of companies that will turn your Photoshop mockup into code. You’ll also find innumerable tutorials that demonstrate how to do it yourself.

This might leave you with the impression that every web project starts with a Photoshop mockup and, through some mystical voodoo, ends up in a web browser as beautifully crafted HTML and CSS. This is a half truth and I’d like to dispel some of misconceptions.
